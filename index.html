Tentang Kami

Rutinitas Pemain Roblox adalah sebuah komunitas Roblox yang berasal dari Indonesia dan salah satu entitas yang menyatukan komunitas-komunitas Indonesia di platform ini!

Survival Mix

Survival Mix adalah mode Survival yang dipadukan dengan fitur-fitur tambahan, seperti plugin custom, ekonomi, rank system, atau elemen RPG.

Skyblock Mode

Skyblock Mode adalah mode di mana player memulai di sebuah pulau kecil di langit, dengan sumber daya yang sangat terbatas. Tujuan dari mode ini adalah untuk bertahan hidup, memperluas pulau, dan menyelesaikan berbagai challenge.

FAQ Server

Pertanyaan yang Sering Diajukan di server Rutinitas Pemain Roblox.

Bagaimana cara register di Lobby?
Caranya sama saja seperti Anda sedang membuat sebuah akun seharusnya formatnya seperti ini /register (Password) (UlangiPassword), Sebagai Contoh /register ikanasin ikanasin artinya password akun anda adalah ikanasin. Dan saat anda mencoba untuk login hanya gunakan format perintah /login (PasswordAkunAnda).
Bagaimana cara teleport ke World Survival?
Anda bisa berinteraksi dengan NPC Random Teleport atau gunakan perintah /rtp.
Dimana saya bisa melihat perintah yang tersedia di server?
Gunakan ketik perintah /help.
Bagaimana cara membuat laporan atau melaporkan seorang player di server?
Anda bisa menggunakan ketik perintah /report untuk membuka Report GUI, Atau anda bisa menggunakan ketik perintah /reportbug atau /reportplayer (NamaPlayer).
Dimana saya bisa melihat voting server untuk server ini?
Anda bisa melihat semuanya di satu untuk semua Platform Portal link Rutinitas Pemain Roblox di link berikut ini : bit.ly/nikashinetwork
Dimana saya bisa membeli VIP/Kunci Crate?
Anda bisa membelinya di WhatsApp Official Rutinitas Pemain Roblox wa.me/message/UOJWKB7V3Z3DE1 atau di Discord Server kami lalu pergi ke channel Server Support > Purchase Stuff.
Mengapa data player saya disetel ulang ke awal dalam Seasonal Event?
Karena mode Seasonal Event berbeda dari SMP, data player dari season SMP sebelumnya akan dipulihkan di season SMP berikutnya. Pemisahan ini memastikan bahwa data Seasonal Event tidak mempengaruhi gameplay utama SMP.
